Experiment 3 - Generative Methods

Description

We are prototyping a world generator. We want to make a cave-like area for a 2D top down combat based game. Right now, we are just experimenting with different methods.

Technical

Using noise with an added gradient, we can make this open area, surrounded by walls. Each tile uses its location and its seed to determine its value. Then it gains another ammount that directly relates to its distance from the center Then, you can use the smoothing method, which will change each tile's value to the average value of it and its neighboring tiles. Or, you can use the clean up function, which will set the value depending on how many of its neighbors increase the threshold. Though this doesn't achieve the exact effect we were looking for, but it makes for some neat generations when used in combination with the smoothing method.
